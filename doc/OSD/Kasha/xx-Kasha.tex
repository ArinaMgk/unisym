\subsubparagraph{Print Collection} \

\textbf{ConPrintChar}

\textbf{Preset}: ES=VideoBufferSegment(0xB800) STACK

\textbf{Parameter}: $AL/I char > \sim AH/I style=0x07 > \sim BX/I posi=\sim$

\textbf{Return}: $BX=NextPosi(automatic mode)$

\textbf{Description}: Print a char on screen.

If bx is invalid or out of the range, bx will be set 0.

``posi'' starts from 0.

Special chars for automatic position mode: \verb|\n|, \verb|\r|

\textbf{Attribute}: BIT (16) CPU (i8086)

\textbf{Nestcall}: ConCursor (2 versions)

\textbf{ConPrint}

\textbf{Preset}: ES=VideoBufferSegment(0xB800) DS=SegofString STACK

\textbf{Parameter}: $?SI String, AH/I\ Style$

\textbf{Return}: $BX=NextPosi\ Vola\{?AX\}$

\textbf{Description}: Print a char on screen.

Accept \verb|\n|, \verb|\r| for controlling and \verb|\0| for end.

\textbf{Attribute}: BIT (16) CPU (SI:i8086, ESI:386+)

\textbf{Nestcall}: ConCursor (2 versions)
